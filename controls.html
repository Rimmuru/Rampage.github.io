Controls 

 ⌨️ [ On Keyboard ] ⌨️

    Controls with the arrow keys
    
    [ F5 ] - Open Trainer
    [ ^ ] - Up
    [ v ] - Down
    [ < ] - Left
    [ > ] - Right
    [ ← ] - Back
    [ Enter ] - Select
    
    Controls with the numpad keys
    
    [ F5 ] - Open Trainer
    [ Numpad8 ] - Up
    [ Numpad2 ] - Down
    [ Numpad4 ] - Left
    [ Numpad6 ] - Right
    [ Numpad0 ] - Back
    [ Enter ] - Select
    
    
     [ On Controller ] 
    
    Press RB and A - Open Trainer (XBox Controller)
    Press R1 and X - Open Trainer (PS Controller)
    DPAD Up - Up
    DPAD Down - Down
    DPAD Left - Left
    DPAD Right - Right
    B/Circle - Back
    A/X - Select
